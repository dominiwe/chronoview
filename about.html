About

What is this?

The main purpose of this tool is to visualize time tracking and planning information. It can interact with different tools called connectors using their APIs and visualize gathered data.

How does it work?

This app runs client-side in the browser and stores data locally using IndexedDB. It is built with SvelteKit, Tailwind, Event Calendar and Plot.
It uses SvelteKit with adapter-static in order to work with Github Pages.

Which connectors are supported?

Currently supported:

  • None

Support planned:

  • ActivityWatch (in Development)
  • Clockify
  • OpenProject
  • SAP
  • Outlook
